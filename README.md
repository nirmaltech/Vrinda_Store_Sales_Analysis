# Vrinda_Store_Sales_Analysis
🚀 Project Overview

This project is an interactive Sales Analysis Dashboard built using Microsoft Excel. The dashboard provides comprehensive insights into Vrinda Store's sales performance, customer behavior, order trends, and channel performance for the year 2025.

The objective of this project is to transform raw sales data into meaningful business insights that help stakeholders make data-driven decisions.

📌 Business Problem

Vrinda Store wants to understand:

Which months generate the highest sales?
What is the relationship between sales and order volume?
Which gender contributes more to revenue?
What are the most common order statuses?
Which states generate the highest sales?
Which age groups are the most valuable customers?
Which sales channels perform best?

📈 Key Insights
💡 Customer Insights
Women customers contribute significantly more revenue than men.
Adult customers form the largest customer segment.
💡 Sales Insights
Sales show strong performance during peak shopping periods.
Higher order volume directly impacts revenue growth.
💡 Geographic Insights
Maharashtra generates the highest sales revenue.
Southern and Northern states contribute major portions of overall sales.
💡 Channel Insights
Amazon and Flipkart are the leading sales channels.
Marketplace platforms contribute the majority of total orders.
🎯 Business Recommendations
Focus marketing campaigns on adult female customers.
Strengthen partnerships with high-performing marketplaces.
Increase inventory for top-selling regions.
Improve customer retention strategies.
Reduce cancellation and return rates through better order management.
